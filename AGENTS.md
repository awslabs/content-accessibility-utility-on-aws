# AGENTS.md

This repository's guidance for AI coding agents lives in
**[CLAUDE.md](CLAUDE.md)**. Please read and follow it — it is the single source
of truth for build/test commands, the docs-site workflow, diagram tooling, and
project conventions.

`AGENTS.md` exists so agents that look for this filename are pointed to the same
instructions. Do not duplicate guidance here; update `CLAUDE.md` instead.
